DualSlider Component

A dual-handle range slider for selecting min/max values

Example 1: RMS Filter

Range: 0.0 to 1.0, Step: 0.01

Current values: min: 0.00, max: 1.00

Example 2: Duration Filter

Range: 0 to 50 seconds, Step: 0.1

Current values: min: 0.0s, max: 50.0s

Example 3: Balance Filter

Range: -1.0 (left) to 1.0 (right), Step: 0.01

Current values: min: -1.00, max: 1.00

Example 4: Compact Variant

Smaller handles and track

Current values: min: 0.00, max: 1.00

Example 5: Dark Mode Variant

White handles and range on dark background

Current values: min: 20.00, max: 20000.00
