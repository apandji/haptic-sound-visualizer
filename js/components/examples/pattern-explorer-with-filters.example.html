This example demonstrates the PatternExplorerWithFilters variant, which automatically wires FilterPanel changes to PatternExplorer. When you adjust filters, the file list updates automatically.

Try it:

  • Type in the search box to filter by filename
  • Adjust the RMS, Duration, Balance, or Movement sliders
  • Click RESET to clear all filters
  • Click the FILTERS toggle to collapse/expand
  • Click play buttons to preview files
