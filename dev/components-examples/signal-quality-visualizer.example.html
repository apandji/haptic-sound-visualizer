Signal Quality Visualizer Component

This example demonstrates the SignalQualityVisualizer component with mock data from data/ganglion_sample_data.csv.

How it works:

  • Connect Device: Shows the floating widget and starts monitoring
  • Disconnect Device: Hides the widget and stops monitoring
  • Toggle Panel: Expands/collapses the quality table
  • Click the button: Click the floating button (bottom-right) to expand/collapse

Features:

  • Information-dense minimized widget (shows all 4 channels at a glance)
  • Displays RMS values (μV) with color-coded quality indicators
  • Shows device type (e.g., "Ganglion v4 Signal")
  • Expandable panel with detailed quality table
  • Real-time updates every 1 second (configurable)
  • Mock data from CSV file (cycles through data)
  • Quality classification: good (green), ok (orange), poor (red)
  • Each channel can have different quality (realistic variation)

Quality Metrics:

  • RMS (μV): Root mean square amplitude (1-45 Hz band) - shown in minimized view
  • Quality: Classification based on thresholds (shown via color):
    • Good (green): RMS 5.0-100.0 μV
    • OK (orange): RMS > 100.0-150.0 μV
    • Poor (red): Everything else

Minimized View: Shows RMS values (μV) for each channel. Color indicates quality classification.
